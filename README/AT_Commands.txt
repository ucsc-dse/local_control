GSM AT Commands
'''''''''''''''

AT&F: Factory settings


0,1: registered to home network
AT+CREG: Network Registration
    - Displays network registration status.

AT+CGREG: GPRS Network Registration
    



AT+COPS: Operator Selection
    - Registers/displays network operators available.

AT+CPOL: Preferred Operator List
    - Edit preferred list of networks

AT+COPN: Read Operator Names
    - Returns a list of operator names from the modem.

AT+CFUN: Set Phone Functionality
    - Sets level of functionality
    - 0 minimum functionality
    - Full functionality

AT+CPIN: Enter PIN

AT+CSQ Signal Strength
